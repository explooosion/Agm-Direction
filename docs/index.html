Agm-Direction

Agm-Direction is the directive for @agm/core (not official)

  • Angular 2+
  • Google Map API
  • Playground

Feature

  • Remove Direction
  • Show Panel Direction
  • Directions Renderer Options
  • Directions Service
  • Custom Icons

Installation

Installation is done using the npm install command:

  • Install @agm/core

    npm install --save @agm/core

  • Install agm-direction

    npm install --save agm-direction

Importing Modules

  • @agm/core
  • agm-direction
import { BrowserModule } from '@angular/platform-browser';
import { NgModule } from '@angular/core';
import { AppComponent } from './app.component';

import { AgmCoreModule } from '@agm/core';            // @agm/core
import { AgmDirectionModule } from 'agm-direction';   // agm-direction

@NgModule({
  declarations: [
    AppComponent
  ],
  imports: [
    BrowserModule,
    AgmCoreModule.forRoot({ // @agm/core
      apiKey: 'your key',
    }),
    AgmDirectionModule      // agm-direction
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

Usage

  • HTML

    <button type="button" (click)="getDirection()">Get</button>

<agm-map [latitude]="lat" [longitude]="lng">
  <agm-direction *ngIf="dir" [origin]="dir.origin" [destination]="dir.destination"></agm-direction>
</agm-map>

  • CSS

    agm-map {
    height: 400px;
}

    }
  }

Show Panel Direction

Use of the DirectionsRenderer object to display a directions panel.

  • HTML

    <agm-direction ... [panel]="myPanel"></agm-direction>
<div #myPanel></div>

Or you could define a function using the panel:

  • HTML

    <agm-direction ... [panel]="setPanel()"></agm-direction>
<div id="myPanel"></div>

  • TS

    function setPanel(){
  return document.querySelector('#myPanel'); 
}

Directions Renderer Options

This object defines the properties that can be set on a DirectionsRenderer object.

  • HTML

    <agm-direction ... [renderOptions]="options"></agm-direction>

  • TS

    options = {
  suppressMarkers: true,
  draggable: true,
  markerOptions: {
    icon: 'my_marker.png'
  },
  ...
};

Directions Service

You can calculate directions (using a variety of methods of transportation) by using the DirectionsService object.

  1. Driving Options - drivingOptions

  2. Transit Options - transitOptions

  3. HTML

    <agm-direction ... [transitOptions]="transitOptions" [travelMode]="travelMode"></agm-direction>

<!-- or -->

<agm-direction ... [drivingOptions]="drivingOptions"></agm-direction>

  4. TS

    transitOptions = {
      departureTime: new Date('2018/03/20 12:00'),
      modes: ['BUS'],
};
travelMode = 'TRANSIT';
